## 注意

* 登录页的验证码是用 `npm` 的 `captcha` 包实现的
* 这三个登陆页都使用了 `layui` , [Layui官网](https://www.layui.com/),[Github项目页面](https://github.com/sentsin/layui/)
* 后台登陆使用了 `x-admin` , [x-admin官网](http://x.xuebingsi.com/)
